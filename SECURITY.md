# Security Policy

## Supported Versions

Currently, the following versions of Game Dev Studio are supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

We take the security of this project seriously. If you find any security vulnerabilities, please do not report them via public issues. Instead, report them by emailing [jack.vhknguyen@gmail.com].

Please include details about:
* The nature of the vulnerability
* Steps to reproduce or exploit it
* Potential impact

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue promptly.
